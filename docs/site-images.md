# The AI Struggle Bus - Site Images & Illustrations

This document describes all the images and illustrations needed for the website. Each image should follow the design style guide: flat vector illustrations with thick outlines, using the brand color palette (Cream, Van Blue, Mustard, Paper Beige, Coral).

---

## Primary Illustration Style

All illustrations feature the **AI Struggle Bus** as the central character. The style is:
- **Flat vector** with thick black outlines (2-3px stroke)
- **Limited color palette** from brand colors
- **Whimsical but professional** tone
- **Rounded, friendly shapes**
- **No eyes or faces** on the bus (clean, professional look)

### Bus Design Specifications
- **Body colors:** Van Blue (#2872A6) lower panels, Cream (#F5F0D7) upper/window area
- **Roof:** Van Blue (#2872A6)
- **Logo placement:** White logo on blue surfaces, blue logo on light surfaces
- **Windows:** Plain/clear with light blue tint - no cartoon eyes
- **Background:** Cream (#F5F0D7) unless otherwise specified

## Production Notes

### File Formats
- **Web use:** PNG for illustrations, WebP for photos
- **Backup:** Keep source prompts in `docs/aisb-content.md`

### Image Generation
- Model: `gemini-2.5-flash-image` (nano-banana)
- Scripts: `scripts/generate-image.py` (single), `scripts/generate-batch.py` (batch)
- Always reference existing bus images for style consistency
- Include logo image for proper branding placement
- See `docs/aisb-content.md` for all prompts

### Optimization
- All images should be optimized for web
- Use responsive images with srcset where appropriate
- Consider lazy loading for below-fold images

### Accessibility
- All images need descriptive alt text
- Decorative images should have empty alt=""
- Ensure sufficient contrast in illustrations
